# 📱 React Native + Supabase Authentication

Este projeto foi desenvolvido com o objetivo de aprender como implementar um sistema de autenticação utilizando **React Native (Expo)** e **Supabase**.

## 📖 Sobre o projeto

O aplicativo permite que usuários criem uma conta, realizem login e mantenham a sessão autenticada utilizando os recursos de autenticação do Supabase.

O foco principal do projeto foi compreender o funcionamento da autenticação em aplicações React Native, além da integração com um backend utilizando Supabase.

## ✨ Funcionalidades

- Cadastro de usuários
- Login com e-mail e senha
- Logout
- Persistência da sessão
- Redirecionamento após autenticação
- Tratamento de erros de autenticação

## 🛠 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Expo Router
- Supabase
- AsyncStorage
- Ferramentas de IA para suporte no desenvolvimento

## 🚀 Como executar

### Clone o repositório

```bash
git clone https://github.com/SEU_USUARIO/react-native-supabase-auth.git
```

```bash
cd app-supabase
```
